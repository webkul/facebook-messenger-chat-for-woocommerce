WordPress WooCommerce Facebook Chat Messenger facilitates the store owner to allow customers to chat with him.

== Installation ==
1. Upload the 'wp-woocommerce-facebook-chat-messenger folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Configure the plugin using the Facebook Messenger Chat menu.

== Description ==
**WooCommerce Facebook Chat Messenger**  the admin can allow customers to chat with store owner with the help of fb chat messenger. A popup will display over store and customer can start chat.

== Features ==
* Admin can chat live via facebook messenger.
* The admin sets the domain of the page where the admin wishes to display the messenger icon.
* Admin can display the messenger icon on all the frontend pages such as my account, cart page, etc.
* A color for the messenger icon can be chosen by the admin.
* A feasible way to engage with the customers.
* Client handling is much easier.
